Required JSON format

Each question should look like this. choices can be an array of 5 strings or 5 objects with letter and text.

[
  {
    "subject": "Pathophysiology",
    "system": "Cardiovascular System",
    "stem": "A 62-year-old man presents with crushing substernal chest pain...\n\nWhich of the following best explains this patient's hypotension?",
    "choices": [
      "Decreased left ventricular preload",
      "Increased pulmonary capillary wedge pressure",
      "Acute mitral regurgitation",
      "Increased systemic vascular resistance",
      "Decreased right ventricular contractility"
    ],
    "correct": "E",
    "objective": "Identify right ventricular infarction as a cause of preload-dependent hypotension after inferior MI.",
    "rationale": "Right ventricular infarction commonly occurs with inferior MI due to RCA occlusion...",
    "whyWrong": {
      "A": "This is the downstream result, not the primary mechanism.",
      "B": "Pulmonary capillary wedge pressure is typically not elevated in isolated RV infarction.",
      "C": "Acute MR causes pulmonary edema and a new systolic murmur.",
      "D": "SVR may increase compensatorily but does not explain the primary finding."
    }
  }
]

Allowed subjects/systems should match the options already used on the website, e.g. Pharmacology, Pathology, Cardiovascular System, Infectious Diseases.
